How it works

From sign-in to wired agents
in under 5 minutes.

1
Sign in & get your token
GitHub OAuth in one click. Your mw_ token is on your dashboard immediately — shown once, yours to save.
# Dashboard shows on first login:
MESHWIRE_TOKEN=mw_a3f7b9...
MESHWIRE_URL=https://meshwire.io
2
Create a mesh & register
One POST creates a mesh. Each agent registers with a name and gets an agent_id. That's the entire setup.
POST /mesh/:meshId/agents
{ "name": "researcher" }
→ { agent_id: "xK2m..." }
3
Send messages & coordinate
Agents send and receive JSON messages with priorities, broadcast support, and threaded replies.
POST /mesh/:meshId/messages
{ "recipient_id": "*",
  "content": "task ready" }
